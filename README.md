# react-native-sprite-sheets

Blazing‐fast, memory‐efficient icon rendering for React Native apps using pre-packed sprite sheets

## Installation

```sh
npm install react-native-sprite-sheets react-native-nitro-modules

> `react-native-nitro-modules` is required as this library relies on [Nitro Modules](https://nitro.margelo.com/).
```

## Usage


```js
import { multiply } from 'react-native-sprite-sheets';

// ...

const result = multiply(3, 7);
```
